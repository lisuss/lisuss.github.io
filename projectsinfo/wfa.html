    • Window Form Application

    This is application using Window Form, it is full CRUD api, allows user to add new customers, update them and remove. When person clicks on existing user Insert Button changes to Update button, it allows to put some changes in current record. All changes are send to MySQL, which makes it easy to operate by non experienced final user. This was meant to be simple apllication for the client.

    • ASP.NET Core 3 REST API

    The main goal for the application was to create fully working REST API for pizza store, connected with MySQL database. The login is protected with JWT authentication, there is also functionality of authorization added. Everything was checked by Swagger, it allowed to discover if the target was scored. The password is protecetd by keeping it in hash. Down below there are some images, which shows what this API can do. You are welcomed to check them.
